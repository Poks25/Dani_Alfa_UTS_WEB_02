# 🛍️ Alfas Step - Modern Footwear E-Commerce

## 📝 Deskripsi Project
**Alfas Step** adalah platform e-commerce berbasis web yang berfokus pada penjualan sepatu modern, mulai dari **Sneakers, Casual, hingga Formal Shoes**.

Project ini dirancang untuk memberikan pengalaman belanja yang **cepat, ringan, dan interaktif**, tanpa menggunakan backend server. Seluruh data dikelola langsung di browser menggunakan **JavaScript** dan **LocalStorage**, sehingga aplikasi tetap dapat berjalan tanpa koneksi ke database eksternal.

💡 Project ini cocok digunakan sebagai:
- 💼 Portfolio project profesional  
- 🧪 Simulasi sistem e-commerce  
- 🚀 Dasar pengembangan ke aplikasi fullstack

## ✨ Fitur Unggulan

### 🛍️ Pengalaman Belanja
- **Dynamic Product Catalog** → Produk ditampilkan dari file JSON  
- **Product Detail** → Informasi lengkap sebelum membeli  
- **Smart Cart System**:
  - Tambah / hapus produk  
  - Update jumlah barang  
  - Perhitungan total otomatis  
- **Checkout Simulation** → Proses pembelian sederhana  
- **Order History** → Riwayat transaksi tersimpan  

---

### 👤 Sistem User
- Register & Login (Simulasi)  
- Data user disimpan di LocalStorage  
- User wajib login sebelum checkout  

---

### 🎨 UI & User Experience
- Responsive (Mobile, Tablet, Desktop)  
- Tampilan clean & modern  
- Rendering dinamis menggunakan JavaScript  
- Sistem rating untuk meningkatkan kepercayaan pengguna  

---

### 📦 Data & Penyimpanan
- Data produk berasal dari `products.json`  
- Cart, user, dan transaksi disimpan di LocalStorage  
- Tidak membutuhkan database eksternal  

---
